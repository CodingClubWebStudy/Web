웹경험

저는 2006년부터 웹 경험을 했어요. 실무 경험은 아니고 고등학교 재학중에 기능대회를 준비하면서 웹을 했었는데요 10년이면 강산도 바뀐다더니 10년 사이에 웹 트랜드도 엄청나게 바뀌어버려서 놀라고 있는 중입니다. 과거에는 테이블로 레이아웃을 짜고 CSS는 그냥 약간의 조미료 같은 역할이라 생각을 했었는데 2008년 들어서부터 웹 표준에 대해 관심을 가지고 웹 표준을 준수하기 위해 (브라우저는 그렇지 못했지만) 열심히 F5키를 눌러대던 추억이 있네요. 그러다가 2009년에 웹 퍼블리싱 알바도 해보고... 요즘에는 졸업작품 중, 자원 관리 모듈을 웹 어플리케이션으로 작성하고 있는데요. PHP+코드이그나이터로 시스템을 구축하고 있습니다. 그래도 아직 부족한게 많기에 잘 부탁드려요~
